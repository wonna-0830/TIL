## 📝 문제 요약
> 문자열 s를 숫자로 변환한 결과를 반환하는 함수 완성하기
> str이 "1234"이면 1234를 반환
> "-1234"면 -1234 반환

```java
import java.util.*;
class Solution {
    public int solution(String s) {
        int answer = Integer.parseInt(s);
        return answer;
    }
}
```

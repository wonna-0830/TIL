## 📝 문제 요약
> 자연수 N에 대해 N의 각 자릿수의 합을 구해 return

## 💡 풀이 아이디어 
- `/`과 `%`를 사용해 끝자리 추출

## 🧩 구현 코드
```java
import java.util.*;

public class Solution {
    public int solution(int n) {
        int answer = 0;
        int num = n;

        for(int i = 0; i < (int)(Math.log10(n)+1); i++){
            answer += num % 10;
            num /= 10;
        }

        return answer;
    }
}
```
```java
import java.util.*;

public class Solution {
    public int solution(int n) {
        int answer = 0;

         while(n != 0){
            answer += (n % 10);
            n = n/10;
        }

        return answer;
    }
}
```

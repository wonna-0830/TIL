## 📝 문제 요약
> 양의 정수 n이 양의 정수 x의 제곱이라면 x+1의 제곱을 리턴하고, n이 양의 정수 x의 제곱이 아니라면 -1을 리턴하는 함수 만들기

## 💡 풀이 아이디어 
- Math.sqrt 함수를 통해 제곱근 판별

## 🧩 구현 코드
```java
import java.util.*;
class Solution {
    public long solution(long n) {
        long answer = 0;
        long num = (long)Math.sqrt(n);
        if(num * num == n){
            answer = (num + 1) * (num + 1);
        } else {
            answer = -1;
        }
        return answer;
    }
}
```
